#droptest
========

A small project to test Dropwizard and MongoDB.

It uses the following:
* dropwizard
* dropwizard-views
* mustache.java
* mongojack
* mongodb java driver
* mongodb
* maven

The project includes a ProcFile so that you can run the service using [foreman](https://github.com/ddollar/foreman).
